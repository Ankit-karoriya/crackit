# CrackIt – Exam Paper Sharing Platform

CrackIt is a full-stack web platform where students can upload, share, and access previous exam papers.  
The goal of this project is to make exam preparation easier by allowing students to find and share useful exam resources.

The project is built using the **MERN Stack (MongoDB, Express.js, React, Node.js)** and includes authentication with **Email OTP verification**.

---

## Features

- User registration with **Email OTP verification**
- Secure authentication using **JWT tokens**
- Upload and share exam papers
- Browse papers uploaded by other users
- Alert system for user feedback
- Protected routes for authenticated users

---

## Tech Stack

### Frontend
- React
- React Router
- Axios
- Tailwind CSS
- Context API (State Management)

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- Nodemailer (Email OTP service)
- JWT Authentication
- bcrypt (Password hashing)
